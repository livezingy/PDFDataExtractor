# 使用说明 | Usage Guide

## 基本操作
1. 启动程序，进入主界面
2. 添加 PDF 或图片文件，可批量拖拽
3. 设置输出目录、参数（如检测模式、导出格式、页码等）
4. 点击“开始处理”
5. 处理完成后，可在 Images 标签页预览检测结果
6. 所有数据和可视化图片保存在 output 目录

## 参数说明
- Camelot Mode: 选择表格检测模式（lattice/stream/network/hybrid）
- Export Format: 选择导出格式（CSV/JSON）
- Pages: 选择全部或自定义页码（如 1-3,5）
- Output Folder: 选择输出目录

## 常见问题
- 建议使用高质量、未加密的 PDF
- 若检测结果不理想，可尝试切换模式或调整参数
- 日志和错误信息可在 output path/debug/app.log 查看

# Usage Guide (English)

## Basic Steps
1. Launch the app and enter the main interface
2. Add PDF or image files (batch drag & drop supported)
3. Set output folder and parameters (detection mode, export format, pages, etc.)
4. Click "Start Processing"
5. After processing, preview results in the Images tab
6. All data and preview images are saved in the output directory

## Parameter Description
- Camelot Mode: Table detection mode (lattice/stream/network/hybrid)
- Export Format: Output format (CSV/JSON)
- Pages: All or custom page ranges (e.g., 1-3,5)
- Output Folder: Output directory

## FAQ
- For best results, use high-quality, non-encrypted PDFs
- If results are unsatisfactory, try switching modes or adjusting parameters
- Logs and errors can be found in output path/debug/app.log
