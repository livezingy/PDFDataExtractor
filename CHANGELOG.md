# 变更日志

本文档记录项目的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [2.0.0] - 2025-12-12

### 新增

- ✨ **Streamlit Web界面**：全新的Web界面，替代原有的GUI
- ✨ **PaddleOCR集成**：集成PaddleOCR引擎，支持OCR和表格检测
- ✨ **引擎选择功能**：图像文件可选择PaddleOCR或Transformer引擎
- ✨ **智能界面切换**：根据文件类型自动切换选择选项
- ✨ **模块化架构**：提取器和引擎模块化设计，易于扩展
- ✨ **Streamlit Cloud支持**：支持一键部署到Streamlit Cloud，提供PDFPlumber、Camelot、PaddleOCR试用
- ✨ **环境检测**：自动检测Streamlit Cloud环境，调整功能可用性

### 改进

- 🔄 **删除GUI**：移除PySide6 GUI，专注于Streamlit界面
- 🔄 **优化参数计算**：基于页面特征动态调整参数范围
- 🔄 **统一统计量使用**：优先使用mode，回退到min
- 🔄 **改进错误处理**：更好的错误提示和处理机制

### 修复

- 🐛 **PaddleOCR layout参数**：修复PPStructure不接受layout参数的问题
- 🐛 **静态方法调用**：修复PageFeatureAnalyzer静态方法调用问题

### 文档

- 📚 创建快速开始指南
- 📚 创建Streamlit用户使用指南
- 📚 创建常见问题FAQ
- 📚 创建部署指南
- 📚 更新所有相关文档

### 技术细节

- 重构代码为模块化架构
- 实现工厂模式管理提取器和引擎
- 实现懒加载机制优化启动性能
- 统一接口设计便于扩展
